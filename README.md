Bigine
===

[大好网站](http://dahao.de)中文 AVG 引擎。基于 GPL-3.0 协议开源。

诚挚欢迎广大爱好者对现有功能进行扩展开发，共同构建好的开发社区，为我们的作者和玩家们提供更多更酷更有意思的玩法。

```

	内容：

		设置时间（晨）
		设置人物：滕灿灿，喜
```
